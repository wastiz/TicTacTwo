QSEE multi-CASE Help

This page contains the help information for the QSEE multi-CASE tool. The diverse nature of this tool means that the help has been split into a number of sub-sections to allow the user to identify the appropriate help information for their needs. To view the pages for each sub-tool directly from this page click one of the hyperlinks shown in 'the guide'.

All sub-tools within QSEE multi-CASE include context sensitive help, which provides instant hyper-links to appropriate sections within the help. To activate the context sensitive help from within a model simply press the F1 key or select the "Context Sensitive Help" option from the Help menu.
