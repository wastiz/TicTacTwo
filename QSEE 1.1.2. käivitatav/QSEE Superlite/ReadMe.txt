Registry Information
-----------------------------------------

QSEE-SuperLite does not write any registry values.

The only value read (if present) is "DefaultDir" value from the registry key -

HKEY_LOCAL_MACHINE\SOFTWARE\qSEE Technologies\qSEE SuperLite\Settings


If this value is present then it is used as the directory when the user first selectes "Save As"

If this value cannot be read or is "" then the current dir is used instead.
